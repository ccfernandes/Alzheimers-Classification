Welcome to the Alzhiemer's Disease Classification Diagnostic Support Tool. The goal of this tool is to aid in the diagnosis of a patient using a machine learning algorithm along with various clinical parameters that you have likely already collected from the patient. Note that this model should NOT be used as a standalone tool to diagnose a patient, and should rather be used as an additive to a physician's expertise.

For transparency, we have provided information about the machine learning model and dataset that is currently being used for this tool below. State total num of patients, and how many in each of the diagnostic groups. also early and late MCI is treated as one group (combined)

About the Model

  • Classification algorithm: Decision Tree
  • Using Python's Scikit Library
  • Missing training data was imputted using a KNN (K-Nearest Neighbor) algorithm

Dataset

  • The model was trained using ADNI's GO and 2 datasets.
  • Early and Late Mild Cognitive Impairment has been collapsed into one category (MCI)
  • Using longitudinal patient data
  • Number of Patients Represented: 1,194
  • Breakdown of Patients By Class:
  • Cognitively Normal: 359
  • Mild Cognitively Impairment: 673
  • Alzheimer's Disease: 162
